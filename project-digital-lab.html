Digital_Lab

Next-Gen Electric Circuits Lab Simulator. A real-time, interactive simulation environment built specifically for students to design, visualize, and master complex circuits.

The Challenge

Existing logic simulators were either too complex (requiring downloads and local installation) or too simple (lacking realistic propagation delays). The goal was to build a highly performant, browser-based simulator that engineering students could use to validate their designs instantly.

The Solution

We engineered a custom graph-traversal engine using JavaScript and the HTML5 Canvas API. The engine supports arbitrary wire routing, dynamic gate creation (AND, OR, NOT, XOR), and real-time state propagation at 60 FPS. By avoiding standard DOM elements and rendering directly to Canvas, the application can handle thousands of gates without dropping frames.

Tech Stack

Vanilla JavaScript HTML5 Canvas Node.js Vite
